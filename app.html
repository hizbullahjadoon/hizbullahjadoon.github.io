A Semi-Supervised Learning Approach for Image Classification

Thesis on efficient multi-class image classification

Completed: February 2023
Machine Learning, Computer Vision
Academic Research

Project Overview

Designed a novel multi-class image classification system using binary classifiers for pseudo-label prediction, eliminating the need for oversampling in the traditional One-vs-All approach. The methodology applied a high confidence threshold (0.9) to ensure precise and reliable pseudo-label assignments during training, resulting in significant efficiency improvements.

Python TensorFlow Scikit-learn OpenCV Matplotlib

Key Features

  • Binary classifier-based pseudo-label prediction
  • High confidence threshold (0.9) for reliable assignments
  • Elimination of oversampling requirements
  • 5 reduction in model size
  • Optimized for resource-constrained environments

Project Details

Challenge

Traditional multi-class classification approaches requiring extensive computational resources and large model sizes.

Solution

Developed a semi-supervised approach using binary classifiers with high-confidence pseudo-labeling.

Results

Achieved 5 reduction in model size and 40% faster inference while maintaining 94% classification accuracy.
